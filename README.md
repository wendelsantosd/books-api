# Overview

This is a simple API, written in Typescript, framework Nest JS and uses the MongoDB.

# Contribute

1. Clone the repository:

```shell
git clone https://github.com/wendelsantosd/books-api
```

2. Enter the books-api directory:

```shell
cd books-api
```

3. Install dependencies:

```shell
npm i
```

or

```shell
yarn
```

4. Start development server:

```shell
npm run start:dev
```

or

```shell
yarn start:dev
```

The last step will run the API with watch, enabling hot reload.
